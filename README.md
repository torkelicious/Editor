```
controls:
HJKL to move (Arrow keys in INSERT mode)
I: Enter Insert mode 
A: Append (enter Insert after cursor) 
X: Delete char on cursor
D: Delete Line
O: Insert into NewLine
Q: Quit
ESCAPE: Exit Insert mode
Undo/Redo with U / R
You can navigate quickly with TAB and SHIFT+TAB (or W and B)

```
